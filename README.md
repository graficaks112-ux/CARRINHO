Carrinho de Compras

O carrinho de compras é a parte do sistema responsável por armazenar os produtos que o usuário seleciona. 
  Nele é possível adicionar itens,remover produtos e visualizar os itens escolhidos, facilitando o controle da compra 
antes da finalização.
O projeto foi desenvolvido utilizando os princípios da Programação Orientada a Objetos (POO). 
  Dessa forma, os elementos do sistema são organizados em classes e objetos, onde cada classe representa 
uma parte do sistema, 
  como produtos ou o próprio carrinho de compras.
Com a orientação a objetos, o código fica mais organizado, reutilizável e fácil de manter, pois cada classe possui 
suas próprias responsabilidades e métodos para executar as ações do sistema. Assim, o carrinho consegue gerenciar 
os produtos adicionados pelo usuário de forma estruturada e clara.
